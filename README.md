# educational-practice-2024
Учебные проекты по предметам учебная и производственная практика со второго семестре второго курса.

3 WPF-приложения и перечень отчётов, содержащих небольшие задания для Windows Forms, исходные файлы которых не сохранились.

Функционал в приложениях: чтение и запись информации с текстовых файлов, вывод в Excel. Итоговая работа может не запускаться из-за отсутствия адаптивности к расположению проекта.

Какого-либо форматирования в отчётах нет. Имена замазаны в целях конфиденциальности.
