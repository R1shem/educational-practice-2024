# educational-practice-2024
Учебные проекты про предметам учебная и производственная практика со втором семестре второго курса.

3 WPF-приложения и перечень отчётов, содержащих небольшие задания для Windows Forms, исходные файлы которых не сохранились.

Функционал в приложениях: чтение и запись информации с текстовых файлов, вывод в Excel.

Какого либо форматирования в отчётах нет, как и требования к нему на тот семестр. Имена замазаны в целях конфиденциальности.
